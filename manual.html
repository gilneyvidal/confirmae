Manual de uso do Confirmaê

Aprenda como acessar o painel, configurar o evento, cadastrar convidados, enviar convites, usar QR Code, operar a portaria e exportar listas.

Acessar painel Acessar portaria
Importante

Guarde seus dados de acesso

Você receberá pelo WhatsApp o ID do evento, o PIN do painel e o PIN da portaria.

Não misture os PINs

O PIN do painel é para administrar o evento. O PIN da portaria é apenas para validar entradas.

1

Acesso do anfitrião

Use o link do painel, o ID do evento e o PIN do painel para administrar seu evento.

2

Cadastro de convidados

Cadastre convidados, acompanhantes, telefone e e-mail. O sistema gera link e QR Code individual.

3

Confirmação online

Cada convidado confirma ou recusa presença pelo próprio link recebido.

4

Portaria com QR Code

No dia do evento, a equipe usa o PIN da portaria para validar entradas pelo QR Code.

Primeiro passo

1. Dados de acesso

Após a liberação do evento, você receberá pelo WhatsApp:

ID

ID do evento

É o identificador do seu evento dentro do Confirmaê.

PIN

PIN do painel

Usado pelo anfitrião para editar o evento e cadastrar convidados.

QR

PIN da portaria

Usado apenas pela equipe da entrada para validar convidados.

Atenção

Não compartilhe o PIN do painel com a equipe da portaria. A portaria deve usar apenas o PIN próprio dela.

Painel

2. Como acessar o painel do anfitrião

Abra o link do painel enviado pelo WhatsApp. Depois informe:

  • ID do evento: o código do seu evento.
  • PIN do painel: a senha de administração enviada pelo WhatsApp.
  • Botão: clique em “Acessar evento”.

Se os dados estiverem corretos, o painel será liberado.

Configuração

3. Como configurar o evento

No painel, preencha os dados principais do evento:

  • Nome do evento: exemplo: Aniversário da Maria.
  • Tipo do evento: casamento, aniversário, festa, formatura ou outro.
  • Nome do anfitrião: nome de quem está organizando.
  • Data oficial do evento: usada para controle interno e acompanhamento.
  • Data exibida no convite: texto que aparece para o convidado.
  • Horário: horário de início do evento.
  • Local: endereço ou nome do espaço.
  • Mensagem do convite: texto de apresentação para os convidados.

Depois clique em “Salvar dados do evento”.

Convidados

4. Como cadastrar convidados

Na área “Novo convidado”, preencha:

  • Nome do convidado: campo obrigatório.
  • Telefone: opcional, mas recomendado.
  • E-mail: opcional.
  • Acompanhantes permitidos: quantidade liberada para aquele convidado.

Depois clique em “Criar convite”.

Limite de convidados

Cada plano possui um limite. Se o limite for atingido, o sistema bloqueia novos cadastros e informa que é necessário solicitar upgrade.

Envio

5. Como enviar convite

Após cadastrar um convidado, o sistema gera um convite individual. Você pode usar:

  • Copiar link: copie o link e envie pelo WhatsApp.
  • Ver QR: abre o QR Code individual do convidado.
  • Baixar QR Code: salva a imagem do QR Code.

Cada convidado possui um link e um QR Code próprio.

Confirmação

6. Como o convidado confirma presença

O convidado abre o link recebido e escolhe uma das opções:

✓

Confirmar presença

O status muda para “Aceito”.

×

Recusar convite

O status muda para “Recusado”.

O painel do anfitrião atualiza os status automaticamente.

Portaria

7. Como usar a portaria

No dia do evento, a equipe da entrada deve abrir o link da portaria e informar:

  • ID do evento: o mesmo ID usado no painel.
  • PIN da portaria: senha própria da equipe de entrada.

Após liberar a portaria, é possível validar convidados de duas formas:

  • Câmera: ler o QR Code do convidado.
  • Validação manual: colar o link ou token do convite.

Quando o convidado é validado, ele é marcado como “Presente”.

Exceção

8. Liberação manual na portaria

Se um convidado recusou o convite, mas chegou ao evento, a portaria pode liberar manualmente.

Status especial

Quando isso acontece, o sistema marca como “Presente com liberação manual”.

Isso ajuda a manter o controle correto da entrada.

Relatórios

9. Como exportar a lista de convidados

No painel do anfitrião, você pode exportar:

  • Todos: lista completa.
  • Confirmados: apenas convidados que aceitaram.
  • Presentes: pessoas que entraram no evento.
  • Recusados: convidados que recusaram.

O arquivo é baixado em CSV e pode ser aberto no Excel, Google Planilhas ou enviado para a equipe do evento.

Boas práticas

10. Dicas importantes

  • Cadastre nomes claros: isso facilita a organização e a portaria.
  • Confira acompanhantes: defina corretamente quantos acompanhantes cada convidado pode levar.
  • Não compartilhe o PIN do painel: ele permite editar o evento.
  • Use o PIN da portaria: envie para a equipe da entrada apenas o PIN da portaria.
  • Teste antes do evento: valide um QR Code de teste antes da entrada começar.
  • Exporte a lista: baixe uma cópia antes do evento.
Suporte

Precisa de ajuda?

Envie uma mensagem no WhatsApp informando o ID do evento, o que você tentou fazer e, se possível, um print da tela.
